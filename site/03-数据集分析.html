    1.1 读数据并统计分类数量¶

    读取文本文件中的数据，分析了数据的基本信息，包括显示前10行数据、输出总样本数量以及每个类别的样本数量。使用了 pandas 库来处理数据，并使用 Counter 进行样本类别的计数

    # 读取数据
content = pd.read_csv('./data/data/train.txt', sep='\t')
# 打印前10行
print(content.head(10))
# 获取样本数量
print(len(content))
# 统计每个类别的数量
count = Counter(content.label.values)
# 打印信息
print(count)
print(len(count))
print('***************************************')

    1.2 分析样本分布¶

    统计了样本的总量、每个类别样本数量的比例，并计算了每行样本的长度以及文本长度的均值和方差。具体步骤如下：

    1. 计算样本总量： 通过遍历每个类别的样本数量，累加得到样本总量。
    2. 计算每个类别样本数量的比例： 遍历每个类别，计算其样本数量在总量中的比例，并输出。
    3. 统计每行样本的长度： 使用 pandas 库的 apply 函数，计算每行样本的长度，并将结果添加到数据框中。
    4. 统计文本长度的均值和方差： 使用 numpy 库计算文本长度的均值和方差，并输出结果

    具体实现如下：

    # 统计样本总量
total = 0
# 遍历每个类别的样本数量
for i, v in count.items():
    total += v
print(total)
# 计算每个类别样本数量的比例
for i, v in count.items():
    print(i, v / total * 100, '%')

print('***************************************')
# 统计每行样本的长度
content['sentence_len'] = content['sentence'].apply(len)
# 打印前十行结果
print(content.head(10))
# 统计文本长度的均值和方差
length_mean = np.mean(content['sentence_len'])
length_std = np.std(content['sentence_len'])
print('length_mean = ', length_mean)
print('length_std = ', length_std)

    1.3 进行分词预处理¶

    使用了结巴分词库对文本进行分词，并对分词结果进行处理和保存。具体步骤如下：

    1. 结巴分词： 定义了一个函数 cut_sentence 用于进行结巴分词。
    2. 添加分词结果列： 使用 apply 函数将分词结果添加到数据框的新列中，列名为 'words'。
    3. 分词： 使用 apply 函数，将每行文本进行结巴分词，并将结果连接为字符串。 对分词结果进行处理，只保留每行的前30个元素。
    4. 结果存放到 CSV 文件： 将处理后的结果保存到名为 'train_new.csv' 的 CSV 文件中。

    具体实现如下：

    # 结巴分词
def cut_sentence(s):
    return list(jieba.cut(s))
# 添加一列数据存储分词后的结果
content['words'] = content['sentence'].apply(cut_sentence)
# 打印前十行
print(content.head(10))
# 分词
content['words'] = content['sentence'].apply(lambda s: ' '.join(cut_sentence(s)))

# 将分词后的结果只保留30个元素
content['words'] = content['words'].apply(lambda s: ' '.join(s.split())[:30])
# 结果存放到csv文件中
content.to_csv('./data/data/train_new.csv')
